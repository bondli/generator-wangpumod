### 如何开始

```bash
tnpm install
```


### 浏览器端运行模块

```bash
gulp
```

然后会自动打开浏览器：http://localhost:8000 （然后选择你对应的端进行预览和开发）

### 发布前需要打包

```bash
gulp build
```
